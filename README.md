## 📂 Repositórios e Projetos Acadêmicos

Abaixo estão os links diretos para cada repositório individual contendo o código-fonte completo e o detalhamento técnico de funcionamento:

### 1. ❌ [AppJogoDaVelha](https://github.com/thiagofjau/AppJogoDaVelha) ⭕
* **Descrição:** Lógica clássica de jogo em tabuleiro 3x3 com gerenciamento de turnos e mapeamento dinâmico de componentes.
* **Principais Conceitos:** Layout fluido com `Grid`, manipulação de eventos com `sender`, e controle de estado em matrizes bidimensionais (`string[,]`).

### 2. 🏨 [MauiAppHotel](https://github.com/thiagofjau/MauiAppHotel) 🛏️
* **Descrição:** Assistente de simulação e contratação de reservas com cálculo dinâmico de diárias.
* **Principais Conceitos:** Mecanismo de `Data Binding` (`BindingContext`), controles de entrada (`DatePicker`, `Stepper`) e navegação de telas com `PushAsync`/`PopAsync`.

### 3. ☀️ [MauiAppTempoAgora](https://github.com/thiagofjau/MauiAppTempoAgora) 🌧️
* **Descrição:** Consulta meteorológica em tempo real baseada na cidade digitada ou na localização atual via GPS.
* **Principais Conceitos:** Consumo de APIs REST com `HttpClient`, parsing de JSON com `Newtonsoft.Json`, serviços nativos de `Geolocation` e exibição de mapas via `WebView`.

### 4. 🛒 [MauiMinhasCompras](https://github.com/thiagofjau/MauiMinhasCompras) 📊
* **Descrição:** Gerenciamento e controle de listas de compras offline com atualização e buscas em tempo real.
* **Principais Conceitos:** Banco de dados local com `SQLite`, operações CRUD assíncronas, sincronização automática de interface com `ObservableCollection<T>` e recurso *Pull-to-Refresh*.

### 5. 🏋️‍♂️ [AppFit](https://github.com/thiagofjau/AppFit) 🏃‍♂️
* **Descrição:** Diário fitness completo para monitoramento e registro de treinos e atividades físicas.
* **Principais Conceitos:** Padrão arquitetural **MVVM**, desacoplamento de lógica com `ICommand`, navegação estruturada via `Shell` (`TabBar`) e padrão de projeto *Singleton* para o banco de dados.
